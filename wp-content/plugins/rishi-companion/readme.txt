== Description ==
Rishi Companion is a plugin that offers powerful features for the Rishi Theme. It includes features to speed up your website and fine-tune it.

== Key Features ==

**_The Rishi Companion includes the following features:_**

* Customizer Reset: It allows you to reset your customizer to default settings quickly.
* Sticky Header: It allows you to add a sticky header to your website.
* Transparent Header: It allows you to add a transparent header to your website.
* Performance: It includes the settings to disable unwanted settings and boost your website speed quickly.
* Reading Progress Bar: Add a reading progress bar on your website to show your readers their reading progress.
* Sidebar Blocks: It allows you to add blocks like Recent Posts, Popular Posts, Author Bio, and many more to the sidebar.
* Social Sharing: It allows you to add a social sharing option to your single posts.

**_Rishi Companion is packed with performance-boosting features such as:_**

* Smart Lazy Load for images with loading animation
* Remove Version parameters from CSS and JS files
* Remove default WP Emoji files
* Disable self-pingbacks
* Disable RSS Feeds
* Defer JavaScript files to improve the loading speed

= Minimum Requirements =

* WordPress 6.3 or greater
* PHP version 7.4 or greater

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select rishi-companion.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download rishi-companion.zip
2. Extract the rishi-companion directory to your computer
3. Upload the rishi-companion directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard
